TXSQL is a MySQL kernel branch maintained by the OpenTenBase team. It is 100% compatible with native MySQL versions and provides many features similar to MySQL Enterprise Edition, such as enterprise-level transparent data encryption, auditing, dynamic thread pools, encryption functions, backup and recovery, parallel query, etc. TXSQL has not only made a lot of optimizations to the InnoDB storage engine, query optimization, replication performance, etc., but also improved the usability and maintainability of the cloud database MySQL. While providing users with all the functions of MySQL, it also provides enterprise-level disaster recovery, recovery, monitoring, performance optimization, read-write separation, transparent data encryption, auditing and other advanced features.
